# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-06

### Added
- **Smart Planner** - Task orchestration with queue management
- **Worker Pool** - 5 worker types (Builder, Researcher, Documenter, Verifier, Evaluator)
- **Experience System** - XP, reputation, and knowledge sharing
- **Policy Engine** - Execution, Safety, and Quality policies
- **Connection Manager** - Multi-LLM support (OpenAI, Anthropic, DeepSeek)
- **Human Gate** - Review cards for human-in-the-loop decisions
- **Dashboard API** - Project tracking and observability
- **Command Interface** - Natural language command parsing
- **FastAPI Server** - REST API backend
- **React Frontend** - Web dashboard UI
- **CI/CD** - GitHub Actions workflows

### Test Coverage
- 350 tests passing (100%)
- Integration tests for Planner + Workers
- Policy engine execution tests

## [0.0.1] - 2026-03-04

### Added
- Initial project structure
- Core data schemas (Idea, Project, Task)
- Basic Planner loop prototype
